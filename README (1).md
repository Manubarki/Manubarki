## `❯ projects.list()`

> *Tired of spending hours screening profiles and repeating the same X-ray searches weekly. So I automated it all.*

### 🐙 [repo-extract](https://github.com/Manubarki/repo-extract) `TypeScript`
Search any GitHub repo · Extract up to ~2,000 contributors · Enrich profiles with company, location, socials & emails · Filter by location · Export to CSV. Surfaces engineers who ship real code — beyond LinkedIn.

### 🧩 [LinkedinProfileScorer](https://github.com/Manubarki/LinkedinProfileScorer) `JavaScript`
Chrome extension that sits on top of LinkedIn and LinkedIn Recruiter · Auto-scores candidate profiles against your hiring criteria · Highlights relevant keywords · Cuts profile screening time dramatically.

### 🗺️ [SourcingCompass](https://github.com/Manubarki/SourcingCompass) `JavaScript`
AI-powered talent mapping tool · Input a role → get target companies with match scores · Poachability signals (layoffs, equity, growth stage) · Built for sourcers who want to hunt smarter, not harder.

### ⚡ AI Fit Scoring Chrome Extension `Claude + Gemini`
Scores candidate fit against job requirements · Explains reasoning · Highlights keywords on any ATS · Plugs into your existing workflow.

### 🧠 System Design Interview Evaluator `Glean Agent`
Reviews interview transcripts · Outputs Hire / No Hire · Rubric-based ratings across 6 dimensions incl. AI skills · Level signal from P4–P7 with confidence scores.

### ⚙️ Automated Google X-ray Engine `Google Apps Script`
Runs daily in the background — zero manual effort · Appends fresh profiles to pipeline · Removes duplicates · Pipeline grows while you sleep.
